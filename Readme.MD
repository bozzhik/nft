# NFT store

The project for my NFT collection, created using React and Tailwind CSS, Vite

### Getting Started

1. Clone the repository

```bash
git clone https://github.com/bozzhik/nft.git
```

2. Open the repository folder

```bash
cd nft
```

2. Install dependencies

```bash
npm i
```

3. Run server

```bash
npm run dev
```
