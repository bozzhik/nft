# NFT store

The project for my NFT collection, created using React and Tailwind CSS, Vite

### Getting Started

1. Clone the repository

```bash
git clone https://github.com/bozzhik/nft.git
```

2. Open repository folder

```
cd nft
```

3. Install dependencies

```bash
npm install
```

4. Run server 

```bash
npm run dev
```
